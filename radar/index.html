工具 github.com/chopratejas 收藏于 2026-06-19

headroom↗

AI Agent 的「上下文压缩层」：prompt、工具输出、日志、RAG、对话历史送进 LLM 之前先压一道，省 60–95% token 而答案不变。三种接入方式——库调用、本地 proxy 零改码、MCP server，另有 headroom wrap claude·codex·cursor 一键包裹。实现是 Python 编排 + Rust 引擎：统计分析器判定 JSON 数组可压性后做抽取式 k-split、AST 级代码压缩、可逆压缩（原文落本地、prompt 里只留 hash 标记，模型要时再取回）、以及护住 provider KV-cache 前缀的告警器。Apache-2.0，约 3.7 万 star。
项目 github.com/OpenBMB 收藏于 2026-06-15

VoxCPM2↗

OpenBMB（MiniCPM 团队）开源的 2B 参数 Tokenizer-Free TTS，扩散自回归架构：支持 30 语种 + 自动语种识别、用自然语言描述做「声音设计」、可控音色克隆（情感 / 语速 / 表达）、48kHz 录音棚级音质，4090 上 RTF≈0.3 实时流式。Apache-2.0，代码与权重全开源，29.5k star。
框架 github.com/omnigent-ai 收藏于 2026-06-14

Omnigent↗

Databricks 的 Matei Zaharia 团队开源的多 agent meta-harness（Apache-2.0）：统一界面架在 Claude Code、Codex、Pi 及自定义 agent 之上，用「策略」而不是 prompt 做管控，live session 跨终端 / Web / 手机同步并支持实时协作，本地优先、无云依赖。

为什么收藏 · 做「不止 Claude Code」的多 agent 系统时，可复用的编排、管控与跨端协作底座。
